## Plain Text Version (for Reference)

### Code Availability

All simulations and metrics were implemented in the open-source `spinalmodes` package (v0.1.0). Source code, public API (`compute_countercurvature_metric`, `geodesic_curvature_deviation`, `compute_scoliosis_metrics`, `solve_beam_static`), and reproducible experiment scripts (`experiments/countercurvature/*`) are available at <your GitHub URL>. A minimal API is exposed via `spinalmodes.countercurvature.api`, and a 30-line demo is provided in `examples/quickstart.py`. Quickstart commands: `python examples/quickstart.py`, `python -m spinalmodes.experiments.countercurvature.experiment_spine_modes_vs_gravity --quick`, `python -m spinalmodes.experiments.countercurvature.experiment_phase_diagram --quick`.

### Data Availability

All figures and numerical outputs in this study are generated by the scripts in `experiments/countercurvature/` and saved automatically to `outputs/`. Raw CSVs (e.g., `outputs/experiments/spine_modes/spine_modes_results.csv`, `outputs/experiments/phase_diagram/phase_diagram_data.csv`, `outputs/experiments/microgravity/microgravity_results.csv`) and derived figures (e.g., `outputs/figs/fig_countercurvature_metrics.png`, `outputs/experiments/phase_diagram/phase_diagram.png`) are reproducible by running the provided commands. No external or proprietary datasets were used.
